##  Genetic Algorithm Simulation 
**Features:**

 - Predator
 - Elitism 
 - Natural Disaster 
 - Mutation 
   - Bit Flip
   - Inversion 
   - Random Set
 - Gene length and Size
 - GeneType
  - Amino Acid
  - Nucleotide
  - Binary
 - Variable Feature via GUI
 - Overall Report(Logfile)
 - GUI visualization 
 - Multiple Creatures
 - Dominating species 
 - Mutualism 
 - Communalism
